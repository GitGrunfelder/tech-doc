HTML

Hypertext Markup Language

<div class='something'></div>

Assign tags and attributes of those tags, often giving tags specific classes to act on with CSS.

<body></body>
  • Skeleton of the page
  • Bones on which CSS act
  • Must be semantic for accessibility
  • Simple to learn
  • Hard to master

CSS

Cascading Style Sheets, the flesh on the bones of HTML. There are layers with specificity that creates the layout of your page.

body { background-color:black; }

Above is an example that would change the background color of the body of the html. Below, when you hover your mouse, the color of the text changes.

a:hover {color:green;}

Python

A versatile high-level programming language used by various companies for a wide variety of tasks. Machine learning, AI, data, webscraping, automation, and even frontend capabilities.

print('Python is easy to read and understand.')

new_list = [i for i in array if i % 2 == 0]
